Quick-Start Guide

What you need to know to get started with Oberon right away

Welcome! Here's a quick tutorial that will tell you all the basics for how to get Oberon up and running quickly.

Installation

Download the Oberon zip file to wherever you want the files to live. Make sure that Microsoft .NET Framework 3.5 is installed on your system.

You're all set!

Files for Oberon

The bot has a few files and directories that need to be present in order to run correctly. Make sure the below are all present.

Main files:

  • Oberon.exe
  • Plugins/System/Core.dll
  • Config/Log.config

Supporting libraries:

  • DeviantArt.Chat.Library.dll
  • log4net.dll
  • Microsoft.Practices.Unity.dll
  • Microsoft.Practices.ObjectBuilder2.dll

Configuration

If this is the first time you've configured the bot, follow the instructions below. Otherwise, see the normal config instructions.

1st-time configuration:

  1. Double-click Oberon.exe
  2. Enter your bot username. This must be a real dA account because it will be used to login to the chat network. It can be any existing dA account, even your own.
  3. Enter your bot password. This must the real password for the bot account.
  4. Enter your bot trigger. A 'trigger' is a string the bot looks for during a chat so that it knows it needs to do something.
  5. Enter the bot owner. This is the dA account of the user responsible for running the bot. This person will have super-admin privileges to the bot.
  6. Enter auto-join channels. Add a chatroom name that you want to the bot to join. Whenever the bot starts up it will join these rooms. Note: these can be changed later.
  7. Dance! You're all done!

Normal configuraton:

There's nothing in this section that you are required to change. But you can if you want! After running for the first time the bot will save some config files. In the /Config directory open Bot.config.

Here you can modify all the settings you put in the first time: username, password, trigger, autojoin rooms, etc. You can also change the about string so that when a user asks about the bot, it will respond with that information.

And that's all the configuration you need.

Summary

The bot is very simple to get up and running. Just make sure you have all your requirements, all your files, run the bot executable, enter some very basic data and BAM you're off!

For more detailed instructions about how to use the bot and configure it's behavior, check out the User Manual.
